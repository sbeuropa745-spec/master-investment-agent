# OPERATIVE MEMORY

## Función

La memoria operativa almacena:

- decisiones estratégicas
- eventos relevantes
- cambios mercado
- señales importantes
- errores detectados
- aprendizajes del sistema

---

## Objetivos

Permitir que el sistema:

- mantenga continuidad
- recuerde decisiones anteriores
- evite errores repetidos
- consolide experiencia
- adapte estrategias

---

## Tipos de memoria

### Memoria estratégica

Guardar:

- cambios importantes
- escenarios mercado
- narrativa dominante
- objetivos activos

---

## Memoria operativa

Registrar:

- señales detectadas
- alertas generadas
- oportunidades relevantes
- riesgos críticos

---

## Memoria histórica

Mantener:

- eventos mercado
- movimientos extremos
- resultados estrategias
- comportamiento agentes

---

## Prioridades iniciales

1. continuidad sistema
2. seguimiento decisiones
3. análisis eventos críticos
4. aprendizaje progresivo
5. contexto persistente

---

## Filosofía

Un sistema inteligente debe recordar, comparar y evolucionar.
